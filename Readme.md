### 6.哈夫曼树的实现与应用
- **构建和使用（如何创建结构体）**  
构建哈夫曼树时，首先需要确定树中结点的构成。由于哈夫曼树的**构建**是从叶子结点开始，不断地构建新的父结点，直至树根，所以结点中应包含指向父结点的指针。但是在**使用**哈夫曼树时是从树根开始，根据需求遍历树中的结点，因此每个结点需要有指向其左孩子和右孩子的指针。
